## CSV flow

1. Export your Karuta sheet.
2. Run the import command (or ask an admin) — Akira fuzzy-matches on character + series.
3. Anything ambiguous stays unclaimed so you can fix names manually.

After import, use [lookup](/commands/lookup) to sanity-check names against Akira’s pool and [collection](/commands/collection) to confirm what landed in your binder. New players should still read [Your first card](/getting-started/your-first-card).
