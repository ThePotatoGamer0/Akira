---
title: Server buff pool
description: Daily communal pot that can unlock temporary server-wide buffs when thresholds are met.
---

The **server buff pool** is your guild’s shared bits pot. Once a day, Akira can roll for server-wide buffs when combined contributions cross tier thresholds (see the reference table below on this page). New to bits? Start with [Bits — the only currency](/economy/bits).

Contributions usually count toward **[personal milestones](/economy/personal-milestone)** as well — the private track and the communal pool move together.
