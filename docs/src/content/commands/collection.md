Akira will help with **/collection** — Optional filters later via flags; the WebUI shows the full grid. Pair with [lookup](/commands/lookup) when you want to browse the pool without owning yet.

## Parameters

Parameters are still settling in for slash commands; check in-game `/help` for the live option list. Akira keeps descriptions short so slash autocomplete stays readable.

## Example usage

```
/collection
```

## Tips

- In Discord you get a browsable binder; the WebUI is better for scanning a large collection in a grid.
- Filters (when available) stack with how you remember cards — series, tags, or favourites vary by how your server uses Akira.
