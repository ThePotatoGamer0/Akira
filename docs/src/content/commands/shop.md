Akira will help with **/shop** - Items include drop luck, grab protection, and burn value boosts. You pay in [bits](/economy/bits); this is separate from [server pool](/economy/server-pool) buffs.

## Parameters

Parameters are still settling in for slash commands; check in-game `/help` for the live option list. Akira keeps descriptions short so slash autocomplete stays readable.

## Example usage

```
/shop
```

## Tips

- Buffs from **/shop** spend [bits](/economy/bits) and are personal - they are not the same as [server pool](/economy/server-pool) rolls or [milestone](/economy/personal-milestone) rewards.
- Items are temporary; note duration when you buy so you do not burn bits right before a buff you already have is about to expire.
